დავალება 2

HTML დოკუმენტში შექმენით ღილაკი ტექსტით Show Message.
ღილაკზე დაჭერისას კონსოლში დაიბეჭდოს ტექსტი: "Button pressed".

HTML დოკუმენტში შექმენით input ელემენტი.
მომხმარებლის მიერ მნიშვნელობის შეყვანისა და ველის დატოვების შემდეგ (blur-ის ივენთი) კონსოლში დაბეჭდეთ შეყვანილი ტექსტი.

შექმენით ცვლადი score და მიანიჭეთ რიცხვითი მნიშვნელობა.
პირობითი ოპერატორის გამოყენებით შეამოწმეთ, არის თუ არა ქულა 50-ზე მეტი ან ნაკლები.
დაბეჭდეთ შესაბამისი შეტყობინება:

"Passed" — თუ პირობა სრულდება

"Failed" — როცა არ სრულდება

ციკლის გამოყენებით დაბეჭდეთ 1-დან 30-მდე ყველა რიცხვი, რომელიც 3-ზე უნაშთოდ იყოფა.
